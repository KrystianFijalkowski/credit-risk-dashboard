# Credit Risk Dashboard

A full-stack web application for analyzing the credit risk of a loan portfolio.
Built to explore how banks measure and visualize risk: default rates, exposure,
and how risk differs across customer segments.

> **Status:** 🚧 Work in progress — built step by step as a learning + portfolio project.

## What it does (planned)

- Stores a portfolio of loans (borrower income, loan amount, segment, repayment status, ...).
- Computes core credit-risk metrics: **default rate**, total & average **exposure**,
  risk broken down by segment.
- Presents everything in an interactive **dashboard** with charts and filters.
